# Sivaji Popuri — Portfolio Website

This is a clean single-page portfolio (HTML/CSS/JS) with Mermaid diagrams.

## Quick start (local)
Just open `index.html` in a browser.

## Deploy to GitHub Pages
1. Create a repo (e.g., `sivaji-portfolio`)
2. Upload these files to the repo root
3. GitHub → Settings → Pages
4. Source: `Deploy from a branch` → `main` → `/ (root)` → Save
5. Your site will be live in ~1 minute

## What you must edit
Search for `REPLACE` and update:
- GitHub / LinkedIn URLs
- Resume link
- Metrics (throughput, cost reduction, stability, etc.)

## Add screenshots
Create an `assets/` folder and drop images there.
Then add `<img>` tags in each project card.
